This file main.py contains two endpoints
1) Finalise the restaurant decision for groups
2) Generating user specific restaurant recommendations

Currently deployed and accessible at https://yapaloysius.pythonanywhere.com 
with the routes: /decision and /recommendations.

**Available until 1st Feb 2023**

To run the files, ensure to run the command "pip install requirements.txt" in terminal

To generate/update restaurant data,
1. Create an account at https://www.yelp.com/signup
2. Access the .env file under data_collection, and pass in the client_id and api_key
3. Run generate_restaurants_singapore.iypnb
